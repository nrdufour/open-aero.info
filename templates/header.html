OpenAero

OpenAero is a catalog for any resources related to aviation, whatever it's a dataset, a software or manuals.

This project has been sleeping for far too long since 2005. It's related to the GNU/Linux Aviation HOWTO.


It's now being rebuilt using Refuge and Couchapp to allow anybody to replicate the dataset and reformat it as it fits.
